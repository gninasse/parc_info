# Tech Stack

## Backend
- **Programming Language:** PHP 8.2+
- **Framework:** Laravel 12
- **Architecture:** Modular (nwidart/laravel-modules)
- **Key Packages:**
  - `spatie/laravel-permission` for RBAC
  - `spatie/laravel-activitylog` for auditing
  - `tightenco/ziggy` for routing

## Frontend
- **Framework:** TailwindCSS v4
- **Bundler:** Vite
- **UI Components:** AdminLTE integration

## Database
- **ORM:** Eloquent ORM
