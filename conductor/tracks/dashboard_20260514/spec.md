# Track: Implement central dashboard for IT Asset Management

## Overview
Develop a central dashboard for IT Administrators to view high-level metrics and perform quick actions regarding IT assets and human resources.

## Goals
- Provide a summary of all active assets (hardware, software, workstations).
- Display recent activity logs.
- Show quick links to manage users and permissions.

## Key Features
- **Asset Summary Widgets:** Total workstations, unassigned assets, etc.
- **Recent Activity Feed:** A list of the latest actions performed in the system.
- **Quick Actions:** Buttons to add new assets or assign them to employees.

## UI/UX
- Use AdminLTE widgets (info boxes, charts) for data-dense and professional presentation.
- Follow TailwindCSS v4 guidelines.
